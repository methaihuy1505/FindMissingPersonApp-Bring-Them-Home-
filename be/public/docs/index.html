Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {your-jwt-token}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

Lấy token từ API /login, sau đó dán vào ô Authorize phía trên theo định dạng: Bearer {token}

Báo cáo vi phạm

API dành cho việc gửi và quản lý các báo cáo vi phạm liên quan đến hồ sơ.

Gửi báo cáo * Người dùng gửi nội dung báo cáo vi phạm cho một hồ sơ cụ thể.

requires authentication

Example request:
curl --request POST \
    "http://localhost:8000/api/reports" \
    --header "Authorization: Bearer {your-jwt-token}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"user_id\": 1,
    \"missing_person_id\": 5,
    \"report_message\": \"Hồ sơ có chứa thông tin sai lệch.\"
}"

const url = new URL(
    "http://localhost:8000/api/reports"
);

const headers = {

  • Xác thực người dùng và trả về JWT token.
Example request:
curl --request POST \
    "http://localhost:8000/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"user@example.com\",
    \"password\": \"password123\"
}"

const url = new URL(
    "http://localhost:8000/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "user@example.com",
    "password": "password123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Thông tin cá nhân (Me)

requires authentication

  • Lấy thông tin chi tiết của người dùng đang đăng nhập từ Token.
Example request:
curl --request GET \
    --get "http://localhost:8000/api/me" \
    --header "Authorization: Bearer {your-jwt-token}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8000/api/me"
);

const headers = {
    "Authorization": "Bearer {your-jwt-token}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};


fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
